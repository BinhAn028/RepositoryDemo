🌍 Digital World Clock Demo

This demo showcases a real-time digital clock component that displays the current time across multiple time zones simultaneously. Built with modern web technologies for optimal performance and user experience.

✨ Features

  • Real-time updates every second
  • Support for 5 major time zones (UTC, EST/EDT, PST/PDT, JST, GMT+7)
  • Toggle between 12-hour and 24-hour formats
  • Responsive design for desktop and mobile
  • Proper timezone handling including daylight saving time
  • Beautiful gradient design with smooth animations
  • Accessibility support with ARIA labels
  • Clean, modern typography
🚀 View Live Demo

🛠️ Technical Specifications

  • Framework: Next.js 15 with React 19
  • Language: TypeScript for type safety
  • Styling: CSS with responsive design principles
  • Time Handling: Intl.DateTimeFormat API for robust timezone support
  • Performance: Optimized with proper interval cleanup
  • Accessibility: ARIA labels and semantic HTML
